This file describes the steps to jump from one application to another using STM32CubeIDE.

- In application project:
1. Change flash start address & size in flash/RAM linker scripts.
2. Enable IRQ in the main function (use __enable_irq() )

- In bootloader project:
1. Change flash size in the flash/RAM linker scripts.
2. Write your bootloader logic
3. Use the bootloader_jump_to_application function.
